What are the different ways to select an element in the DOM?
There are some way to select an element in the DOM they are , getElementById() uses for select an id selector , getElementsByClassNames() uses for an Class Selector, querySelector() uses for a single css element but it support any type of selector in css also querySelectorAll() uses for multiple element select in at a time .
